MODULO 4: Aspectos de diseño Categoría 2

Tipografía accesible

La fuente y el tamaño del texto deben ser legibles. La elección de la fuente, tamaño, interlineado y formato del texto afecta directamente a la legibilidad y comprensibilidad del contenido, especialmente para personas con dislexia, baja visión o dificultades de aprendizaje.

¿Qué significa?

  • Fuentes claras y legibles. Sans-serif para pantalla, caracteres bien diferenciados.
  • Tamaño mínimo 12-14pt. Preferiblemente 14-16pt para contenido principal.
  • Interlineado adecuado.  Mínimo 1.5 veces el tamaño de fuente.
  • Ancho de línea apropiado.  Izquierda, sin justificación completa (crea espacios irregulares).

Características de tipografía accesible:

1. Tipo:

Recomendadas (sans-serif para pantalla):

    • Arial
    • Verdana
    • Calibri
    • Open Sans
    • Roboto
    • Montserrat
    • Lato


Menos recomendadas:

    • Times New Roman (serif, menos legible en pantalla).
    • Comic Sans (infantil, poco profesional, aunque paradójicamente ayuda a algunos con dislexia).
    • Fuentes decorativas o script.

Evitar:

❌ Fuentes con adornos excesivos.
❌ Fuentes ultra finas o ultra gruesas.
❌ Fuentes con caracteres similares difíciles de distinguir.

2. Tamaño:

  • Texto general: 14-16pt (mínimo absoluto 12pt)
  • Texto grande (títulos): 18pt o superior
  • Pie de imagen o notas: mínimo 11pt (evitar si es posible)
  • Para alumnado con baja visión: considerar 16-18pt

3. Interlineado :​

(espacio entre líneas)

  • Mínimo: 1.5 veces el tamaño de fuente
  • Óptimo para párrafos: 1.5-2.0
  • Beneficio: Mejora significativamente la legibilidad y reduce fatiga visual

4. Ancho de línea:​

  • Óptimo: 50-75 caracteres por línea.
  • Máximo: 100 caracteres.
  • Problema de líneas muy largas: Cansan la vista, dificultan encontrar inicio de siguiente línea.
  • Problema de líneas muy cortas: Rompen el ritmo de lectura, obligan a saltos constantes.

5. Espaciado:​

  • Entre párrafos: Espacio suficiente (1.5-2 líneas de alto).
  • Entre palabras: No muy comprimido.
  • Entre letras (tracking): No excesivamente amplio ni comprimido.

6. Alineación:​

  • Preferible: Alineación a la izquierda (más natural para lectura en español)
  • Evitar: Justificación completa (crea "ríos" de espacios irregulares que dificultan lectura)
  • Centrado: Solo para títulos cortos o elementos destacados

7. Formato y énfasis:​

✓ Uso moderado de negritas para énfasis (palabras clave)
✓ Evitar TEXTO TODO EN MAYÚSCULAS (dificulta lectura, percibido como gritar)
✓ Evitar itálicas extensas (menos legibles, usar solo para títulos de obras o términos extranjeros)
✓ Subrayado solo para enlaces (no para énfasis)

Consideraciones especiales:

Para dislexia:

  • Fuentes específicas: OpenDyslexic, Dyslexie (diseñadas para dislexia)
  • Mayor interlineado: 2.0 o superior
  • Evitar justificación: Solo alineación izquierda
  • Fondos suaves: Crema, beige o colores pasteles (no blanco puro que deslumbra)
  • Evitar texto en itálica: Más difícil de leer para personas con dislexia

Para baja visión:

  • Tamaños mayores: 16-18pt mínimo.
  • Alto contraste: Negro sobre blanco o texto muy oscuro sobre muy claro.
  • Evitar combinaciones complejas: Simplicidad visual.

Implementación en eXeLearning:

✅ Configurar estilo con fuentes accesibles: Elegir tema con tipografía sans-serif clara.​

✅ Establecer tamaños mínimos: Verificar que el texto base no sea menor de 14pt.​

✅ Revisar que el tema elegido sea legible: Probar con diferentes dispositivos y tamaños de pantalla.​

✅ No modificar excesivamente la tipografía: La consistencia es clave para la accesibilidad.
