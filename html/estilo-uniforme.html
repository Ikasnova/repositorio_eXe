MODULO 4: Aspectos de diseño Categoría 2

Estilo uniforme

Texto

El REA debe mantener coherencia visual en todo el recurso.

¿Qué significa?

  • Misma familia tipográfica en todo el recurso. Usar 1-2 fuentes como máximo en todo el REA.
  • Paleta de colores consistente. 3-4 colores principales usados sistemáticamente.
  • Formato de títulos y subtítulos homogéneo.  Misma jerarquía visual en todas las páginas.​
  • Estilo de imágenes coherente. Fotografías realistas vs ilustraciones, colores saturados vs pastel.

Elementos de la identidad visual de vuestro REA:

1. Tipografía:

  • Máximo 1-2 fuentes: Una para títulos (puede ser más expresiva), otra para texto (debe ser muy legible).
  • Tamaños jerarquizados: Título (H1) 24-28pt > Subtítulo (H2) 18-20pt > Subtítulo nivel 2 (H3) 16pt > Texto 14pt.
  • Fuentes sans-serif para pantalla: Más legibles en digital (Roboto, Open Sans, Montserrat, Lato).
  • Fuentes serif solo para títulos: Si queréis dar elegancia (Merriweather, Playfair Display).

2. Paleta de colores:

  • 3-4 colores máximo: Principal (60%), secundario (30%), acento (10%).
  • Alto contraste texto-fondo: Negro sobre blanco o gris muy oscuro sobre blanco garantiza legibilidad.
  • Uso consistente del color: Mismo color siempre para actividades, otro para evaluación, otro para recursos adicionales.
  • Herramientas para crear paletas: Coolors.co, Adobe Color, Canva Color Palette Generator.

3. Estilo de eXeLearning:​

  • Mantener el mismo estilo visual en todo el REA (INTEF, Simple, Kahurangi...).
  • Personalizar solo si mejora la experiencia: No por capricho. La personalización debe tener propósito pedagógico.
  • Coherencia en botones, cuadros, destacados: Los iDevices deben mantener el mismo aspecto visual.

4. Formato de iDevices:​

  • Mismo estilo para elementos similares: Todas las actividades con el mismo iDevice, todas las reflexiones con otro.
  • Iconografía coherente: Si usáis iconos, que sean del mismo pack visual.
  • Estructura repetida: El alumnado debe reconocer patrones y saber qué esperar.


Beneficios de la uniformidad visual:

- Aspecto profesional: Transmite que el recurso está cuidado y es fiable.​

- Facilita la navegación: El alumnado aprende rápidamente a orientarse.​

- Reduce carga cognitiva: No tienen que interpretar diferentes códigos visuales constantemente.​

- Mejora la experiencia de usuario: Todo fluye de forma natural y predecible.​
